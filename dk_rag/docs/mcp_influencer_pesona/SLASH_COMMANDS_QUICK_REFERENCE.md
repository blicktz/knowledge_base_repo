## How to Use

1. **Activate a persona:**
   ```
   /dan-kennedy
   ```

2. **Ask your question:**
   ```
   How do I write a sales email that converts?
   ```

3. **Get authentic advice** in the persona's voice with:
   - Their unique catchphrases
   - Their specialized vocabulary
   - Their communication style
   - Real examples from their content

## Multi-Language Support

The system automatically detects your query language:

**English:**
```
/dan-kennedy
How do I write effective copy?
```

**Chinese:**
```
/nihaixia
如何看手相？
```

Response will be entirely in the detected language while maintaining the persona's style.

## Tips for Best Results

1. **Be specific** - Include context about your industry, stage, or constraints
2. **Ask follow-ups** - Build on previous responses for deeper insights
3. **Request examples** - Ask for concrete examples or case studies
4. **Try creative tasks** - Ask them to write, create, or draft something for you

## Regenerating Skills

If persona data is updated:
```bash
make generate-skills
```

Then restart Claude Code to reload the Skills.
